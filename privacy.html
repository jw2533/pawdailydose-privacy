Privacy Policy

Last updated: April 17, 2026

About Paw Daily Dose Youtube

Paw Daily Dose Youtube is a personal automation tool that uploads video content to the developer's own YouTube channel. It is not a public-facing application and does not serve external users.

Google User Data

Paw Daily Dose Youtube accesses the YouTube Data API using the youtube.upload scope solely to upload videos to the developer's own YouTube channel. No other Google user data is accessed, collected, or stored.

Data Collection

This application does not collect, store, or share any personal information from any third party. The only data handled is:

— OAuth tokens required to authenticate with the YouTube Data API, stored locally on the developer's private server.
— Video files created by the developer for upload to YouTube.

Data Sharing

This application does not share any data with third parties. No analytics, tracking, or advertising services are used.

Data Retention

OAuth tokens are retained only as long as needed for the application to function. They can be revoked at any time via Google Account Permissions.

Security

OAuth credentials and tokens are stored securely on a private server and are never exposed publicly or committed to version control.

Contact

For questions about this privacy policy, contact: pawdailydose@gmail.com
